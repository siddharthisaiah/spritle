Postcard

Let people know what intersting things you have been upto by sending them a postcard.

README

'Postcard' is a web app that lets you share the interesting things in your life

Features inlcude:

  • Creating a new postcard
  • Commenting on postcards created by other users
  • Ability to 'like' or 'unlike' a postcard
  • Update the current users Profile information (display pic and bio)

Postcard is built with Django 2.0 and Bootstrap 4 and uses a third party django app - crispy-forms - to display forms.

Crispy forms can be installed by running the following command
>>> pip install django-crispy-forms
